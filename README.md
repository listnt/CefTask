# CefTask

Стэк: CEF, OpenGL, FFmpeg и EGL.  
Написать оконное приложение, которое будет поделено на две части –
верхнюю и нижнюю. В верхней части рендерится CEF в режиме OSR (без своего
окна браузера). В нижней части – рендерится видео с помощью FFmpeg. Кадры
отрисовываются текстурами на OpenGL. EGL используется как создатель окна.  
Веб-страница должна содержать простую анимацию, написанную на JS. Всё
должно воспроизводиться бесконечно. Должно собираться через cmake.  
Тестовое задание должно собираться и запускаться на ОС Ubuntu 24.04 (wsl2
или нативная)

# Инструкция

1. Скачать cef_binary_138 с официального сайта
2. <B>Распаковать содержимое архива в third_party/cef</B>
3. Собрать проект
```
cmake -DCMAKE_BUILD_TYPE=Debug -B build
cmake --build build --target CefTask
```
4. Запустить проект
```
cd build/Debug
./CefTask
```

# Содержимое
```
src/ - содержит исходный код
shaders/ - содержит файлы шейдеров
assets/ - содержит медиа файлы
```
